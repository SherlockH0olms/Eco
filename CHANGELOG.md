# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-23

### Added
- Initial MVP release for hackathon
- Arm raise exercise with pose detection
- Bubble pop gamification mechanics
- Real-time visual feedback (Green/Yellow/Red)
- Caregiver dashboard with progress tracking
- Local data persistence
- Score and repetition tracking
- Session history and statistics
- Progress charts (7/14/30/90 day views)
- Confetti celebrations for milestones
- Android and iOS support
- Camera-based pose detection using Google ML Kit
- Offline-first architecture

### Technical Highlights
- Flutter 3.0+ cross-platform app
- MediaPipe Pose for skeleton tracking
- Real-time angle calculation for form validation
- Provider state management
- FL Chart for data visualization
- SharedPreferences for local storage

### Known Limitations
- Only arm raise exercise available
- Local storage only (no cloud sync)
- Single user per device
- English language only
- Manual session deletion not implemented
