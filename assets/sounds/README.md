# Sound Assets

Add your sound files here for game feedback:

- `pop.mp3` - Bubble pop sound (when correct rep is detected)
- `success.mp3` - Success sound (milestone achieved)
- `complete.mp3` - Session completion sound

## Free Sound Resources

You can download free game sounds from:
- [Freesound.org](https://freesound.org/)
- [Zapsplat.com](https://www.zapsplat.com/)
- [Mixkit.co](https://mixkit.co/free-sound-effects/)

## Format Requirements

- Format: MP3 or WAV
- Duration: 0.5-2 seconds recommended
- Size: Keep under 100KB for best performance
