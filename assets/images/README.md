# Image Assets

Add your image files here for UI elements:

- `logo.png` - App logo (512x512px)
- `avatar_default.png` - Default user avatar
- `exercise_arm_raise.png` - Icon for arm raise exercise
- `exercise_shoulder.png` - Icon for shoulder rotation

## Icon Requirements

- Format: PNG with transparency
- Size: 256x256px or 512x512px
- Style: Simple, high contrast for accessibility

## Free Icon Resources

- [Flaticon.com](https://www.flaticon.com/)
- [Icons8.com](https://icons8.com/)
